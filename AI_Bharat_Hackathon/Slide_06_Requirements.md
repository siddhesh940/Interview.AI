# Slide 06 — Requirements for the Hackathon

---

## **Functional Requirements**
1. AI-powered mock interview system with real-time voice interaction
2. Automated question generation based on job role, experience & resume
3. Instant AI evaluation with detailed feedback and scoring
4. Multi-mode practice — Technical, Aptitude, HR, DSA Coding
5. Resume parsing with skill extraction using OCR
6. Progress tracking dashboard with performance analytics

## **Non-Functional Requirements**
1. **Scalability** — Serverless architecture handles 1000+ concurrent users
2. **Low Latency** — Sub-2 second AI response time using Edge deployment
3. **Accessibility** — Works on any device with a browser (mobile + desktop)
4. **Security** — Clerk Auth with encrypted sessions, Supabase RLS policies
5. **Availability** — 99.9% uptime via Vercel Edge Network

## **Target Audience**
- College students preparing for campus placements
- Fresh graduates appearing for job interviews
- Working professionals switching careers
- Tier-2 & Tier-3 city candidates with limited access to mentors

## **Impact on Bharat**
- **Democratizes interview preparation** — Free AI-powered coaching for all
- **Bridges urban-rural gap** — No need for expensive coaching classes
- **Multilingual potential** — Can be extended to support Hindi & regional languages
- **Employment readiness** — Improves placement success rate for Indian students
