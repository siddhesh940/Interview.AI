# Slide 05 — Technologies to be used in the solution

---

## **Frontend**
- **Next.js 14** — React-based full-stack framework (App Router, SSR, ISR)
- **React 18** — Component-based UI with hooks
- **TypeScript** — Type-safe development
- **Tailwind CSS** — Utility-first responsive styling
- **Framer Motion** — Smooth animations & transitions
- **shadcn/ui** — Accessible, customizable UI components

## **Backend & API**
- **Next.js API Routes** — Serverless REST API endpoints
- **LangChain** — AI orchestration & prompt chaining
- **WebSocket** — Real-time voice interview communication

## **AI & ML**
- **OpenAI GPT-4o** — Question generation, evaluation, NLP, embeddings
- **Retell AI** — Real-time voice STT/TTS for live AI interviews
- **Tesseract.js** — OCR for resume parsing
- **Web Speech API** — Browser-based speech recognition

## **Authentication**
- **Clerk** — Sign Up, Sign In, session management, OAuth providers

## **Database & Storage**
- **Supabase PostgreSQL** — Relational database (profiles, resumes, results, questions)
- **Supabase Storage** — File storage for resumes & documents

## **Deployment & Hosting**
- **Vercel** — Edge deployment, CDN, serverless functions
- **GitHub** — Version control & CI/CD

## **State Management & Utilities**
- **Zustand** — Lightweight global state management
- **nanoid** — Unique URL/ID generation
